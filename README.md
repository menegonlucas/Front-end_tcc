# Front-end_tcc

Projeto Front-end desenvolvido como Trabalho de Conclusão de Curso (TCC) do curso Técnico em Desenvolvimento de Sistemas.

## Visão geral
Aplicação web para busca e visualização de livros, integrando a API do Google Books com uma interface chamada "Palavraria". O projeto foca em usabilidade, acessibilidade e organização de código front-end.

## Demonstração
A versão hospedada está disponível no GitHub Pages:
https://menegonlucas.github.io/Front-end_tcc/

## Estrutura do repositório
- index.html — Página inicial do projeto.
- web/ — Páginas e recursos de demonstração:
  - [web/googlebks.html](web/googlebks.html) — Interface da Palavraria (busca de livros).
  - [web/profile.html](web/profile.html) — Página de perfil.
  - web/img/ — Imagens usadas no projeto.
  - web/scripts/ — Scripts front-end (ex.: googlebks.js, profile.js).
- styles/ — Estilos CSS (ex.: [styles/googlebks.css](styles/googlebks.css), [styles/index.css](styles/index.css), [styles/profile.css](styles/profile.css)).
- Documentação/ — Documentação completa do projeto.

## Tecnologias
- HTML5, CSS3, JavaScript (Vanilla)
- Google Books API (integração para busca de livros)
- Hospedagem: GitHub Pages / Vercel (integração com backend quando necessário)

## Principais recursos
- Busca por título, autor ou assunto usando Google Books API.
- Exibição de resultados e detalhes do livro.
- Layout responsivo e elementos de acessibilidade básicos.
- Estrutura modular de arquivos para facilitar manutenção.

## Como executar localmente
1. Clonar o repositório.
2. Abrir `index.html` ou [web/googlebks.html](web/googlebks.html) no navegador.
3. Para desenvolvimento com servidor local (recomendado):
   - Usar `Live Server` no VS Code ou um servidor HTTP simples:
     ```sh
     npx http-server . -p 8080
     ```
   - Acessar `http://localhost:8080/web/googlebks.html`.
